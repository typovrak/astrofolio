Un immense merci à <a href="https://github.com/markhorn-dev" target="_blank">Mark Horn</a> pour son template <a href="https://astro.build/themes/details/astrosphere" target="_blank">Astro sphere</a> qui m'a permis de partir sur une base solide, performante et complète pour me concentrer pleinement sur la création de contenu, notamment mes [parcours](/parcours), [projets](/projets) et [écrits](/blog) du site.

<TitleLink number={2} title="Icônes"/>

- <a href="https://fontawesome.com/" target="_blank">Font awesome</a> pour les icônes de marques.
- <a href="https://www.radix-ui.com/icons" target="_blank">Radix icons</a> pour les icônes fonctionnels.
- <a href="https://devicon.dev/" target="_blank">Devicon</a> pour les icônes des outils et technologies.
- <a href="https://github.com/JetBrains/icons/blob/master/src" target="_blank">Jetbrains</a> pour quelques icônes comme celui du bouton changelog dans l'entête de chaque page.

- <a href="https://vercel.com" target="_blank">Vercel</a> pour la mise en ligne des sites Front (comme celui-ci 😉).
- <a href="https://render.com" target="_blank">Render</a> pour le déploiement de mes APIs.
- <a href="https://uptimerobot.com" target="_blank">UptimeRobot</a> pour le monitoring des status de l'ensemble de mes sites et APIs.
- <a href="https://supabase.com" target="_blank">Supabase</a> pour le monitoring des visites du site.
